Buttons

Two variants of buttons are available: normal and outlined. All buttons must have the .btn class. You can customize the color of the button and the text through the classes .btn-{color}, .btn-outline-{color} and .text-{color}

Normal
Outline

Button sizes

Buttons are available in three sizes: normal (.btn), large (.btn .btn-lg), and small (.btn .btn-sm)

Block buttons

To make the button span the entirety of it's container, use the class .btn .btn-block

Text

By default, headers have a Raleway font. All other texts (paragraphs, inputs, links, etc) will have a Montserrat font. Proactive Framework does not provide more fonts, but feel free to customize them with your own CSS.

Forms

Proactive Framework provides an easy way to create responsive forms. In this example you have all you need to create your own forms.

Example:

Code:

      
        <form action="">
          <div class="form-group">
            <label for="name">Name</label>
            <input type="text" name="name" id="name" class="form-control" />
          </div>
          <div class="form-group">
            <label for="email">Email</label>
            <input type="email" name="email" id="email" class="form-control" disabled />
          </div>
          <div class="form-group">
            <label for="exampleFormControlSelect1">Example select</label>
            <select class="form-control" id="exampleFormControlSelect1">
              <option>1</option>
              <option>2</option>
              <option>3</option>
              <option>4</option>
              <option>5</option>
            </select>
          </div>
          <div class="form-group">
            <label for="exampleFormControlSelect2">Example multiple select</label>
            <select multiple class="form-control" id="exampleFormControlSelect2">
              <option>1</option>
              <option>2</option>
              <option>3</option>
              <option>4</option>
              <option>5</option>
            </select>
          </div>
          <div class="form-check">
            <input class="form-check-input" type="checkbox" value="" id="defaultCheck1" />
            <label class="form-check-label" for="defaultCheck1">
              Default checkbox
            </label>
          </div>
          <div class="form-check">
            <input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios1" value="option1" checked />
            <label class="form-check-label" for="exampleRadios1">
              Default radio
            </label>
          </div>
          <div class="form-check">
            <input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios2" value="option2" />
            <label class="form-check-label" for="exampleRadios2">
              Second default radio
            </label>
          </div>
          <div class="form-group">
            <label for="message">Message</label>
            <textarea class="form-control" name="message" id="" cols="30" rows="10"></textarea>
          </div>
          <button class="btn btn-green" type="submit">Submit</button>
        </form>

Images

We provide a couple classes to style images: .img-fluid and .img-thumbnail

Responsive images

Responsive images will resize according to their container. Always occupying the maximum witdh available. Resize your browser to see the effects.

Thumbnail images

Thumbnail images have a default height and width, plus rounded corners. Shadows can be added using the .z-depth{1-5} classes.

Note that, if the image does not have a 1:1 aspect ratio, it will be cropped.

Navbar

Proactive Framework provives customizable, semi-responsive* navigation bars. The following is an example of a complete navigation bar, with its source code.

*Note that, for a totally responsive navbar, a JavaScript solution will be needed.

Local Gym
  • Home
  • About
  • Our Work
  • Contact
      
        <nav>
          <div class="nav-wrapper bg-dark">
            <a href="#" class="brand-logo left text-light"><i class="fas fa-dumbbell logo-image"></i>Local Gym</a>
            <ul id="nav-mobile" class="right">
              <li><a class="text-light" href="#">Home</a></li>
              <li><a class="text-light" href="#">About</a></li>
              <li><a class="text-light" href="#">Our Work</a></li>
              <li><a class="text-light" href="#">Contact</a></li>          
            </ul>
          </div>
        </nav>

Switching navbar content

You can switch the position of the logo and the links by using the .right and .left classes on the a and the ul tags.

Here is an example of the same navbar, after switching the position of the links.

  • Home
  • About
  • Our Work
  • Contact
      
        <nav>
          <div class="nav-wrapper bg-dark">
            <a href="#" class="brand-logo right text-light"><i class="fas fa-dumbbell logo-image"></i>Local Gym</a>
            <ul id="nav-mobile" class="left">
              <li><a class="text-light" href="#">Home</a></li>
              <li><a class="text-light" href="#">About</a></li>
              <li><a class="text-light" href="#">Our Work</a></li>
              <li><a class="text-light" href="#">Contact</a></li>          
            </ul>
          </div>
        </nav>

Sticky navbar

There's also a sticky navbar, which will always stay on top of the page when scrolled past it's position. You can achieve this effect by adding .sticky on the nav tag, like the navigation bar on this site.
